About

TikTok Live Video Statistics shows you View, Like and Comment count from any video on this platform! Often data like View, Like or Comment displayed on TikTok are either incorrect or abbreviated, that's why Livecounts.io Team decided to create this free tool!

Data is directly taken from TikTok's Official API Service and is guaranteed to be as accurate as possible.

To get started simply copy and paste Video's URL into search box on Livecounts.io and you're good to go! Don't know how to obtain Video URL? Check our Guide!

Guide

Unfortunately due to TikTok API Limitations we're forced to use search function by Direct Video URL. Guide below will help you with obtaining one.

1. Go to your favorite creator's profile and click on video

2. At the very bottom you should see option to copy link to this video. Click on "copy" button in order to copy URL

3. Once you have URL copied simply paste it into search box on Livecounts.io and press Search button.
